# AUTOMOD Discord Bot

A feature-rich Discord bot built in Python

## Features
- Auto moderation
- Swear filter
- Welcome messages
- Moderation commands (kick, ban, clear)
- Fun commands (8ball, flip, poll)

## Commands
- !hello - Greets the user
- !ping - Check bot latency
- !clear - Delete messages
- !kick - Kick a member
- !ban - Ban a member
